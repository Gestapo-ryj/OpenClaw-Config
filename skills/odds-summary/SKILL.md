# 比赛赔率总结技能

## 描述
当用户需要总结体育比赛赔率、分析投注市场、比较不同博彩公司的赔率时，使用此技能。适用于足球、篮球、网球、电竞等各类体育赛事的赔率分析。

## 激活条件
当用户提到以下内容时激活此技能：
- 比赛赔率、盘口、水位
- 博彩公司、庄家赔率
- 投注分析、赔率比较
- 胜平负、让球、大小球
- 体育博彩、赛事预测

## 分析指标

### 1. 隐含概率计算
```
隐含概率 = 1 / 赔率 × 100%
总概率 = Σ(各结果隐含概率)
博彩公司利润 = 总概率 - 100%
```

### 2. 价值投注识别
```
价值 = (估算概率 × 赔率) - 1
价值 > 0 表示有投注价值
```

### 3. 赔率变化追踪
- 开盘赔率 vs 当前赔率
- 赔率变化幅度
- 市场资金流向

## 故障排除

### 常见问题
1. **数据不完整**：尝试多个数据源
2. **赔率过期**：检查更新时间，必要时重新搜索
3. **格式不一致**：标准化数据提取流程
4. **网站屏蔽**：使用备用网站或API

### 备用数据源
1. 主流博彩公司官网
2. 赔率比较网站
