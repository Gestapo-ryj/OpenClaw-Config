# 比赛赔率总结技能

一个用于搜索、分析和总结体育比赛赔率的OpenClaw技能。

## 快速开始

1. 将整个 `odds-summary` 文件夹复制到 OpenClaw 技能目录
2. 当用户询问比赛赔率时，技能会自动激活

## 功能特点

- 🔍 **多源搜索**: 从多个博彩网站获取赔率数据
- 📊 **智能分析**: 计算隐含概率，识别价值投注
- 📈 **趋势追踪**: 分析赔率变化和市场情绪
- 🎯 **个性化报告**: 根据用户需求生成不同详细程度的报告

## 使用示例

### 基础查询
```
用户: "今晚曼联比赛的赔率是多少？"
助理: 自动搜索并返回赔率总结
```

### 比较查询
```
用户: "比较一下威廉希尔和Bet365的赔率"
助理: 生成对比表格和分析
```

### 深度分析
```
用户: "分析一下这场比赛的投注价值"
助理: 提供价值投注识别和风险分析
```

## 配置选项

在 `config.json` 中可配置：
- 首选数据源
- 默认博彩公司
- 报告详细程度
- 更新频率

## 数据源

支持的数据源：
1. 主流博彩公司官网
2. 赔率聚合网站
3. 体育数据API
4. 专业体育媒体

## 安全提醒

⚠️ **重要**: 本技能仅提供信息参考，不构成投注建议。请遵守当地法律法规，理性投注。
