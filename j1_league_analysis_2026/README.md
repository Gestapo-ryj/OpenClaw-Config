## 🔍 技术细节

### FlashScore字段含义
| 字段 | 含义 | 说明 |
|------|------|------|
| **AT÷** | **主队常规比分** | 90分钟常规时间的得分 |
| **AU÷** | **客队常规比分** | 90分钟常规时间的得分 |
| AG÷ | 主队最终比分 | 包括加时赛/点球的最终得分 |
| AH÷ | 客队最终比分 | 包括加时赛/点球的最终得分 |

### 数据解析流程
1. **数据提取**: 从FlashScore HTML中提取编码数据
2. **字段解析**: 按字段含义解析比分信息
3. **去重处理**: 基于比赛ID、球队、时间去除重复条目
4. **结果验证**: 用户确认关键比赛比分正确性

### 已知问题与解决
1. **重复数据**: 原始数据包含重复比赛条目，已通过去重解决
2. **比分混淆**: 明确区分常规比分（AT/AU）和最终比分（AG/AH）
3. **数据验证**: 用户确认清水鼓动3-1广岛三箭比分正确
